# Deep Research Briefings

This directory contains briefing documents designed for Gemini Deep Research agents.
Each document is self-contained and provides a specific research agent with the project
context and research questions it needs to survey the literature and internet.

## Purpose

StateBind has completed its core pipeline (12 workstreams, 646 tests, 10x retrospective
enrichment). The goal is now to identify what techniques, data, and ML approaches would
elevate this project to a top-tier publication.
